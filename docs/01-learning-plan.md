# Neo4j Cluster Learning Plan

## Objective

Understand Neo4j clustering before introducing Kubernetes concepts.

## Phase 1: Database Clustering

Topics:

- Neo4j cluster architecture
- Core members
- Leader election
- Followers
- Raft consensus
- Transaction replication
- Quorum
- Failure scenarios


## Phase 2: Kubernetes

Topics:

- Pods
- StatefulSets
- Services
- Persistent Volumes
- Secrets
- ConfigMaps


## Phase 3: GitOps

Topics:

- Helm
- ArgoCD
- Git based deployments


## Phase 4: AKS

Topics:

- Node pools
- Azure storage
- Networking
- Monitoring
- Production HA design
